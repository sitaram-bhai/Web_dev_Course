The internet is a vast global network of interconnected computer networks that communicate with each other using standardized communication protocols. It enables users to access a variety of digital resources and services, including web pages, email, messaging, file sharing, social networking, video conferencing, and online gaming, among others.


A web server is a computer system or software that responds to requests from client computers over the internet or a local network, delivering web pages and other content to users. It hosts and serves web pages, files, images, videos, and other digital content using the Hypertext Transfer Protocol (HTTP) or its secure counterpart, HTTPS. The web server software, such as Apache, Nginx, or Microsoft IIS, listens on a specific port for incoming requests and processes them by retrieving the requested content from the server's file system or database, generating dynamic content on the fly, or forwarding the request to another web server or application server. Web servers can also handle authentication, caching, load balancing, and other tasks to ensure fast, secure, and reliable delivery of web content.

A web page is a document that displays images, texts, videos and other content in the web browser, a website is a collection of webpages that link together.

HTML:

Stands for "Hypertext Markup Language." It is a standard language used for creating and structuring web pages and other online content. HTML provides a set of tags that define the content and layout of a webpage.

CSS:

Stands for "Cascading Style Sheets." It is a style sheet language used for describing the visual presentation and layout of HTML and XML documents. CSS provides a set of rules for styling the fonts, colors, backgrounds, borders, and other elements of a webpage.

JS:

Stands for "JavaScript." It is a high-level programming language used for creating dynamic and interactive web pages and applications. JS allows web developers to add behaviors, events, and effects to web pages, as well as to manipulate and validate data entered by users.

A web browser is a software application used for accessing and displaying web pages and other online content on the internet. It interprets and renders HTML, CSS, and JavaScript code to create a visual representation of a webpage that users can interact with.

Web Hosting

Web hosting is a service where you place your website and files on the hostion companies web server.

Types of web hosting

  1. Shared Hosting
  2. Virtual Private Hosting
  3. Dedicated Hosting
  4. Cloud Hosting

Shared Hosting

Shared hosting is a type of web hosting service in which multiple websites are hosted on a single server and share its resources, such as processing power, memory, and storage space. This is a cost-effective option for small businesses and individuals

Virtual Private Hosting

A VPS is a virtual server with dedicated CPU, memory, and bandwidth resources. It is expansive than shared hosting.

Dedicatd Hosting

Dedicatd Hosting is a hardware server that is dedicated to you only. All hardware, CPU, memory, and bandwidth resources are yours to use. Generally, this option is more expensive than a VPS hosting.

Cloud Hosting

Cloud hosting is a type of web hosting service that utilizes a network of interconnected servers to provide scalable, on-demand computing resources, such as processing power, memory, and storage space, to host websites and applications. Cloud hosting also offers features such as automatic backups, load balancing, and global content delivery networks (CDNs) to enhance performance and security.
